# Kakdela-p2p — Фаза 1 (Base)

Минимальный рабочий P2P-мессенджер (фаза 1) — Compose + Room + ViewModel.

Что внутри:
- Android app (Kotlin + Jetpack Compose)
- Room database (Chat, Message)
- ViewModel и simple UI (список чатов, экран чата)
- GitHub Actions workflow — собирает Debug APK и загружает как артефакт.

Сборка в CI:
- При пуше в main/master workflow соберёт APK и загрузит артефакт.
